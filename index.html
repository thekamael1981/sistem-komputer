Level Dasar - Mengenal Sistem Komputer

Pengertian Sistem Komputer

Sistem komputer adalah gabungan dari tiga komponen utama yang bekerja sama:

Hardware

Bagian yang bisa dilihat dan disentuh

Software

Program yang memberikan perintah

Brainware

Manusia yang mengoperasikan

Analogi Sederhana

Seperti mobil: mesin (hardware), aturan lalu lintas (software), dan sopir (brainware)

Level Menengah - Komponen Detail

Mempelajari fungsi dan jenis-jenis komponen sistem komputer

Hardware (Perangkat Keras)

Unit Pemrosesan
  • CPU: Otak komputer yang memproses data
  • GPU: Memproses grafik dan visual
  • RAM: Memori sementara untuk data aktif
Unit Input
  • Keyboard: Input teks dan perintah
  • Mouse: Navigasi dan kontrol pointer
  • Microphone: Input suara
Unit Output
  • Monitor: Menampilkan informasi visual
  • Speaker: Output suara

Microsoft Word, Chrome, Photoshop

Penghubung antara OS dengan hardware tertentu

Driver printer, driver grafis

Level Dasar - Komponen Inti CPU

Mengenal bagian-bagian utama dalam CPU

Anatomi CPU

ALU

Arithmetic Logic Unit - melakukan perhitungan matematika dan operasi logika

Contoh: 2+3=5, TRUE AND FALSE=FALSE
Control Unit

Pengatur lalu lintas data - mengontrol semua operasi CPU

Seperti polisi lalu lintas di persimpangan
Register

Kotak penyimpanan kecil super cepat di dalam CPU

Seperti meja kerja untuk data yang sedang diproses

Level Menengah - Siklus Kerja CPU

Memahami bagaimana CPU menjalankan instruksi

Siklus Fetch-Decode-Execute

1. FETCH (Ambil)

CPU mengambil instruksi dari memori

Program Counter menunjuk ke alamat instruksi berikutnya
2. DECODE (Terjemahkan)

Control Unit menerjemahkan instruksi

Mengidentifikasi jenis operasi dan operand yang diperlukan
3. EXECUTE (Jalankan)

ALU melakukan operasi yang diminta

Data diproses sesuai dengan instruksi
4. STORE (Simpan)

Hasil disimpan kembali ke memori/register

Output siap untuk instruksi berikutnya

Hierarki Memori

Register
Kecepatan: 1 cycle | Kapasitas: Bytes
Cache L1/L2/L3
Kecepatan: 2-10 cycles | Kapasitas: KB-MB
RAM
Kecepatan: 100+ cycles | Kapasitas: GB
Storage (SSD/HDD)
Kecepatan: Ribuan cycles | Kapasitas: TB

Level Lanjut - Optimasi & Performa

Teknik-teknik untuk meningkatkan performa sistem

Teknik Optimasi CPU

Pipelining

CPU dapat mengerjakan beberapa instruksi secara bersamaan dalam tahap yang berbeda

Meningkatkan throughput tanpa mengurangi latensi
Cache Optimization

Menyimpan data yang sering diakses di memori yang lebih cepat

Mengurangi waktu akses ke memori utama
Branch Prediction

CPU memprediksi instruksi mana yang akan dijalankan selanjutnya

Mengurangi penundaan akibat percabangan program

Faktor Performa Sistem

Clock Speed (GHz)

Kecepatan CPU melakukan operasi per detik

Jumlah Core

Kemampuan multi-tasking dan parallel processing

Cache Size

Kapasitas memori cepat untuk data yang sering diakses

Arsitektur

Desain internal CPU (x86, ARM, dll)

Animasi: Siklus Instruksi CPU

Level Dasar - Konsep Sistem Operasi

Memahami peran dan fungsi dasar sistem operasi

Apa itu Sistem Operasi?

Sistem Operasi adalah "penerjemah" antara user dan hardware komputer

Seperti Manajer Perusahaan

Mengatur semua pekerjaan dan sumber daya

Seperti Rambu Lalu Lintas

Mengatur agar semua proses berjalan tertib

Contoh Sistem Operasi:
Windows macOS Linux Android iOS

Level Menengah - Fungsi OS & Booting

Mempelajari fungsi detail OS dan proses startup komputer

Fungsi Utama Sistem Operasi

Manajemen Proses
  • Menjalankan program secara bersamaan (multitasking)
  • Mengatur prioritas proses
  • Mengalokasikan waktu CPU
Manajemen Memori
  • Mengatur pembagian RAM untuk setiap program
  • Virtual memory untuk program besar
  • Memory protection antar proses
Manajemen File
  • Mengorganisir file dalam folder
  • Kontrol akses dan permission
  • Backup dan recovery
Manajemen Device
  • Mengelola driver perangkat
  • Input/output control
  • Plug and play support

Proses Booting Komputer

1
Power-On Self Test (POST)

BIOS/UEFI memeriksa semua hardware

RAM, CPU, storage, keyboard, dll
2
Firmware Initialization

Mengatur konfigurasi basic hardware

Clock speed, voltage, device detection
3
Boot Loader

Mencari dan memuat sistem operasi

GRUB, Windows Boot Manager, dll
4
Kernel Loading

Inti OS dimuat dan mengambil kontrol

Device drivers, system services
5
User Interface

Desktop/login screen ditampilkan

Siap untuk digunakan user
