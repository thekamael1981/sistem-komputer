Kelas X - Konsep Dasar Sistem Komputer

Memahami pengertian dan komponen dasar sistem komputer

Materi Pembelajaran

1. Pengertian Sistem Komputer

Sistem komputer adalah kombinasi antara perangkat keras (hardware), perangkat lunak (software), dan pengguna (brainware) yang bekerja sama untuk memproses data menjadi informasi yang berguna.

2. Komponen Utama Sistem Komputer

Hardware

Perangkat keras yang dapat dilihat dan diraba secara fisik

  • CPU (Central Processing Unit)
  • Memori (RAM & ROM)
  • Storage (Hard Disk, SSD)
  • Input Device (Keyboard, Mouse)
  • Output Device (Monitor, Printer)
Software

Program atau aplikasi yang menjalankan fungsi tertentu

  • Sistem Operasi (Windows, Linux, macOS)
  • Aplikasi (Microsoft Office, Browser)
  • Driver Perangkat
  • Utility Programs
Brainware

Manusia yang mengoperasikan dan mengelola sistem komputer

  • End User (Pengguna akhir)
  • Programmer
  • System Administrator
  • Network Administrator

Kelas XI - Arsitektur Komputer

Memahami arsitektur dan komponen internal sistem komputer

Materi Pembelajaran

1. Arsitektur CPU

ALU (Arithmetic Logic Unit)

Melakukan operasi aritmatika dan logika

CU (Control Unit)

Mengontrol dan mengatur operasi CPU

Register

Memori berkecepatan tinggi di dalam CPU

2. Proses Input-Proses-Output

Siklus dasar pemrosesan data dalam komputer meliputi tahap pengambilan input, pemrosesan oleh CPU, dan menghasilkan output.

3. Hierarki Memori

Cache Memory

Memori tercepat, kapasitas kecil

RAM (Random Access Memory)

Memori utama, volatile

ROM (Read Only Memory)

Memori permanen, non-volatile

Kelas XII - Sistem Operasi & Teknologi Modern

Memahami sistem operasi, booting, dan teknologi cloud computing

Materi Pembelajaran

1. Sistem Operasi

Sistem operasi adalah perangkat lunak yang mengelola sumber daya komputer dan menyediakan layanan untuk program aplikasi.

Manajemen Proses

Mengatur eksekusi program dan alokasi CPU

Manajemen Memori

Mengatur penggunaan RAM dan virtual memory

Manajemen File

Mengatur penyimpanan dan akses file

3. Virtualisasi & Cloud Computing

Virtualisasi

Teknologi yang memungkinkan satu komputer fisik menjalankan beberapa sistem operasi

Cloud Computing

Penggunaan sumber daya komputasi melalui internet

  • IaaS (Infrastructure as a Service)
  • PaaS (Platform as a Service)
  • SaaS (Software as a Service)
